# Rime
一直在找一个体验不错的输入法来使用，发现小狼毫之后才知道原来输入法个人也可以玩的这么炫酷。他给予你很高的自由度，让你可以根据自己的想法进行定制。

rime backup，小狼毫输入法个人定制使用备份。

## 操作说明

将文件夹的内容直接复制到用户文件夹全部进行替换即可使用。
